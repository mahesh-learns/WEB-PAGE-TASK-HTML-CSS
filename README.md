# 🌐 WebPage Task

A simple and responsive front-end web project developed using **HTML5** and **CSS3**. This project demonstrates the implementation of multiple web pages, including a home page, login page, and forgot password page with a clean and user-friendly interface.

## 📂 Project Structure

```
WebPageTask/
│── WEBPAGETASK.HTML
│── LOGINPAGE FOR WEBPAGE TASK.HTML
│── FORGOT PASSWORD FOR WEBPAGE TASK.HTML
```

## ✨ Features

- 🏠 Responsive Home Page
- 🔐 User Login Page
- 🔑 Forgot Password Page
- 🎨 Clean and Attractive User Interface
- 📱 Responsive Layout
- 🌐 Easy Navigation Between Pages

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open **WEBPAGETASK.HTML** in your web browser.
4. Navigate to the Login and Forgot Password pages using the provided links.

This project was created to practice:

- Responsive web design
- Form creation
- Navigation between multiple pages
